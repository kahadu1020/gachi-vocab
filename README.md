# ガチ英単語

会話でよく使われる英単語・熟語を、頻出順に上から覚えるための静的PWAです。

## 起動

```powershell
node server.mjs
```

ブラウザで `http://localhost:4173` を開きます。

同じ Wi-Fi のスマホから試す場合:

```powershell
node server.mjs --lan
```

表示された `スマホ用: http://...:4173/` をスマホのブラウザで開きます。Windows ファイアウォールが出たら、プライベートネットワークを許可します。

## スマホからいつでも使う

GitHub Pagesで公開すると、スマホ単体でいつでも開けます。手順は [PUBLISH.md](./PUBLISH.md) にまとめています。

## 入っているもの

- 頻出会話語句 80 件のスターターデッキ
- IPA 発音記号、コアのイメージ、使い方、例文、よく出る形
- 未学習、復習、定着、熟語の絞り込み
- ブラウザ内保存の復習スケジュール
- 意味から英語、英語から意味、例文穴埋めの四択確認テスト
- 進捗の JSON 書き出し・読み込み
- PWA マニフェストと Service Worker

## 進捗データ

進捗はブラウザの `localStorage` に保存されます。別端末へ移す場合は、右上の書き出しボタンで JSON を保存し、読み込みボタンから取り込みます。
ブラウザのサイトデータ、Cookie、アプリデータを削除すると進捗も消えることがあります。
